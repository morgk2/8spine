## What the app does

8SPINE combines a full music library experience with online service integrations and local file playback.

### Core functions

- **Library management**
  - Import and organize songs, albums, artists, and playlists.
  - Manage downloaded and cloud-sourced content in one place.

- **Advanced playback**
  - Queue controls, track actions, and background playback.
  - Album/track history, favorites, and playback personalization.
  - Fully offline mode for imported and downloaded media.

- **Lyrics & metadata**
  - Integrated lyrics support and metadata matching from multiple providers.
  - Artwork handling with fallbacks and caching.

- **Discovery & recommendations**
  - Artist/album exploration, suggestions, and recommendation flows.
  - Optional discovery-focused behavior and search helpers.

- **Connected services**
  - Integrations for services like Apple Music, Last.fm, and ListenBrainz for smooth syncing and tracking.
  - Advanced cloud provider connectivity for supported workflows.

- **Modular engine**
  - Module-based architecture for extending streaming/search/cloud behaviors.
  - Module management UI with install, reorder, and source controls.

- **Customization**
  - Theme and player appearance options.
  - Interface settings for animated artwork/background behavior.
